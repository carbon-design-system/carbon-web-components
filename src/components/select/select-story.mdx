The select component allows users to choose one option from a list. It is used in forms for users to submit data.

## Getting started

Here's a quick example to get you started.

### JS (via import)

```javascript
import 'carbon-web-components/es/components/select/index.js';
```

<Description markdown={`${cdnJs({ components: ['select'] })}`} />

### HTML

```html
<bx-select helper-text="Optional helper text" label-text="Select" placeholder="Optional placeholder text">
  <bx-select-item-group label="Category 1">
    <bx-select-item value="all">Option 1</bx-select-item>
    <bx-select-item value="cloudFoundry">Option 2</bx-select-item>
  </bx-select-item-group>
  <bx-select-item-group label="Category 2">

```
