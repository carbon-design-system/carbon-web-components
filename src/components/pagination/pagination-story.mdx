The pagination component is used to switch through multiple pages of items, when only a maxium number of items can be displayed per page.
Can be used in combination with other components like data table.

## Getting started

Here's a quick example to get you started.

### JS (via import)

```javascript
import 'carbon-web-components/es/components/pagination/index.js';
```

<Description markdown={`${cdnJs({ components: ['pagination'] })}`} />

<Description markdown={`${cdnCss()}`} />

### HTML

```html
<bx-pagination page-size="10" start="0" total="100">
  <bx-page-sizes-select slot="page-sizes-select">
    <option value="10">10</option>
    <option value="20">20</option>
    <option value="30">30</option>
  </bx-page-sizes-select>
  <bx-pages-select></bx-pages-select>
</bx-pagination>
```
