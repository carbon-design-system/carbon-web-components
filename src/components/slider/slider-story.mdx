Sliders provide a visual indication of adjustable content, where the user can move the handle along a horizontal track to increase or decrease the value.

## Getting started

Here's a quick example to get you started.

### JS (via import)

```javascript
import 'carbon-web-components/es/components/slider/index.js';
```

<Description markdown={`${cdnJs({ components: ['slider'] })}`} />

<Description markdown={`${cdnCss()}`} />

### HTML

```html
<bx-slider label-text="Slider" max="100" min="0" step="1" value="50"></bx-slider>
```

## With input box

To include an input box, place `<bx-slider-input>` within `<bx-slider>`.

```html
<bx-slider label-text="Slider" max="100" min="0" step="1" value="50">
  <bx-slider-input aria-label="Slider value" type="number"></bx-slider-input>
</bx-slider>
```

## Skeleton

For the skeleton variation, utilize `<bx-slider-skeleton>`.

```html
<bx-slider-skeleton></bx-slider-skeleton>
```
