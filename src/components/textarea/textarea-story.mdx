Text inputs enable the user to interact with and input content and data. This component can be used for long and short form entries.

Textarea is a multi-line variant of text input.

## Getting started

Here's a quick example to get you started.

### JS (via import)

```javascript
import 'carbon-web-components/es/components/textarea/index.js';
```

### HTML

```html
<bx-textarea placeholder="Optional placeholder text" rows="4" cols="50"></bx-textarea>
```

## `<bx-textarea>` attributes and properties

Note: For `boolean` attributes, `true` means simply setting the attribute (e.g. `<bx-textarea autofocus>`) and `false` means not setting the attribute (e.g. `<bx-textarea>` without `autofocus` attribute).
