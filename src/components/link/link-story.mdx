Represents a link. Provided for convenience purpose, you can alternatively use `<a class="bx--link">` with Carbon style in place in light DOM.

## Getting started

Here's a quick example to get you started.

### JS (via import)

```javascript
import 'carbon-web-components/es/components/list/index.js';
```

<Description markdown={`${cdnJs({ components: ['list'] })}`} />

<Description markdown={`${cdnCss()}`} />

### HTML

```html
<bx-link href="https://www.ibm.com"> Link </bx-link>
```

## Paired with icon

Import the desired icon and incorporate with the link text:

```javascript
import Download16 from 'carbon-web-components/es/icons/download/16';

function App() {
  return (
    html`<bx-link href="https://www.ibm.com"> Download ${Download16({ slot: 'icon' })}</bx-link>`
  );

```
