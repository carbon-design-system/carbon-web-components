File uploaders allow users to select one or more files to upload to a specific location.

## Getting started

Here's a quick example to get you started.

### JS (via import)

```javascript
import 'carbon-web-components/es/components/file-uploader/index.js';
```

<Description markdown={`${cdnJs({ components: ['file-uploader'] })}`} />

<Description markdown={`${cdnCss()}`} />

### HTML

```html
<bx-file-uploader helper-text="Only .jpg and .png files. 500kb max file size" label-text="Account photo">
  <bx-file-drop-container accept="image/jpeg image/png" multiple="true">
    Drag and drop files here or click to upload
  </bx-file-drop-container>
</bx-file-uploader>
```
