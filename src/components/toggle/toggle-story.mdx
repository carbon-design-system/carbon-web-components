Toggle is a control that is used to quickly switch between two possible states.
Toggles are only used for these binary actions that occur immediately after the user ‚Äúflips‚Äù the toggle switch.
They are commonly used for ‚Äúon/off‚Äù switches.

Use `small` property/attribute to make it more compact in size, so they can be used in use cases such as data tables.

## `<bx-toggle>` attributes, properties and events

Note: `bx-checkbox-*` events are not fired from `<bx-toggle>`.

Note: For `boolean` attributes, `true` means simply setting the attribute (e.g. `<bx-toggle small>`) and `false` means not setting the attribute (e.g. `<bx-toggle>` without `small` attribute).
