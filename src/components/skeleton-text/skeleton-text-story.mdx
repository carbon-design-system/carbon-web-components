### HTML

```html

```

### Lines

To create multiple lines of skeleton text, set the `type` property and use multiple instances of `<bx-skeleton-text>`.

```html
<bx-skeleton-text type="line"></bx-skeleton-text>
<bx-skeleton-text type="line"></bx-skeleton-text>
<bx-skeleton-text type="line"></bx-skeleton-text>
```
