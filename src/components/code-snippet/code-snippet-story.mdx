Code snippets are small blocks of reusable code that can be inserted in a code file.

## Getting started

Here's a quick example to get you started.

### JS (via import)

```javascript
import 'carbon-web-components/es/components/code-snippet/index.js';
```

<Description markdown={`${cdnJs({ components: ['code-snippet'] })}`} />

<Description markdown={`${cdnCss()}`} />

### HTML

```html
<bx-code-snippet>
  node -v Lorem ipsum dolor sit amet, consectetur adipisicing elit. Blanditiis, veritatis voluptate id incidunt molestiae officia
  possimus, quasi itaque alias, architecto hic, dicta fugit? Debitis delectus quidem explicabo vitae laboriosam!
</bx-code-snippet>
```

## Single line

The Terminal style is for single-line.

```html
<bx-code-snippet>
  node -v Lorem ipsum dolor sit amet, consectetur adipisicing elit. Blanditiis, veritatis voluptate id incidunt molestiae officia
  possimus, quasi itaque alias, architecto hic, dicta fugit? Debitis delectus quidem explicabo vitae laboriosam!
</bx-code-snippet>
```

## Multi line

The Code style is for larger, multi-line code snippets.

```html
<bx-code-snippet type="multi">
  node -v Lorem ipsum dolor sit amet, consectetur adipisicing elit. Blanditiis, veritatis voluptate id incidunt molestiae officia
  possimus, quasi itaque alias, architecto hic, dicta fugit? Debitis delectus quidem explicabo vitae laboriosam!
</bx-code-snippet>
```

## Inline

The Inline style is for code used within a block of text.

```html
<bx-code-snippet type="inline"> node -v </bx-code-snippet>
```

## Skeleton

For the skeleton variation, utilize `<bx-code-snippet-skeleton>`.

```html
<bx-code-snippet-skeleton type="single"></bx-code-snippet-skeleton>
```
